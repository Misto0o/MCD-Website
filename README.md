# Minecraft Dungeons Submissions

This project is a web app where users can submit their Minecraft Dungeons gameplay videos via YouTube links. Submissions include details like player name, platform, mob damage, and mob health.

## Features

- **Submit videos:** Users can submit YouTube links of their gameplay.
- **View submissions:** All submissions are displayed in a live-updating table.
- **Anonymous auth:** Users are signed in anonymously to allow deletion of their own submissions.
- **Delete your own submissions:** Users can remove only the submissions they created.
- **Responsive UI:** Styled with Bulma for a clean and simple interface.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Firebase Firestore & Firebase Authentication (Anonymous)
- **Hosting:** Netlify
